GrowthX Digital Marketing Website

This project is a modern digital marketing website developed using HTML, CSS, and JavaScript. It represents a fictional brand named GrowthX and includes multiple sections such as home, services, about, testimonials, and contact. The website features a clean and visually appealing dark-themed interface along with a fixed navigation bar and smooth scrolling functionality for better user experience. It also includes structured content layouts, service cards, and a contact form to simulate a real business website.

This project was created to demonstrate front-end development skills, including webpage structuring using HTML, styling with CSS, and adding basic interactivity using JavaScript. It reflects an understanding of responsive design concepts, user interface design, and modern web layout practices.
